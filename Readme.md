## 🚀 About This Course

This 10-day intensive program introduces students to:

- Python Fundamentals
- Traditional NLP
- Open-Source LLMs
- Prompt Engineering
- RAG Systems
- AI Agents
- Streamlit Web Applications

This course moves from **Python basics → GenAI → End-to-End AI Applications**.

---

## 📌 Course Phases

### 🔹 Phase 1: Python & Traditional NLP
- Python fundamentals
- Text processing
- Control flow & data structures
- File handling & APIs
- Foundational NLP using nltk

### 🔹 Phase 2: GenAI Frameworks & Building
- Hugging Face transformers
- Connecting to LLM APIs (Groq, Ollama)
- LangChain orchestration
- Retrieval-Augmented Generation (RAG)
- AI Agents
- Streamlit UI development

---

## 🎯 Expected Outcomes

By the end of this course, students will be able to:

- Build Python-based AI applications
- Connect to modern LLM APIs
- Create RAG-based chatbot systems
- Develop AI agents
- Deploy interactive GenAI apps using Streamlit
